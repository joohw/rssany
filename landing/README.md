# RssAny Landing — Next.js App Router marketing site.

## Development

```bash
cd landing
npm install
npm run dev
```

Default dev URL: `http://localhost:28473`.

Public site URL and GitHub links are defined in `src/lib/site.ts`.

## Build & Run

```bash
npm run build
npm run start
```

## Deploy

See repo root `npm run deploy` and `.env.deploy.example`.
